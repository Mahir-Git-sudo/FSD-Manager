# FSD-Manager
This is a tool that allows you to download your files faster using aria2.

How to use:
1. Paste the download link
2. Set locations per file where the files should download
3. Then click on "Start All Downloads"
4. You can open the Console tab or the look the green process bar to see how much downloading is complete

Blocking:
Sometimes Microsoft Defender SmartScreen blocks from running scripts that are downloaded from internet. Just click "More info" and then "Run anyway"

Again sometime a window pops up. Just click on "Run"

If the downloader still don't opens, right click on AutoRun.ps1, "Properties" and check on "Unblock"
